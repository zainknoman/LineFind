# 🔍 LineFind - Text Search Utility

A fast, lightweight **client-side text search tool** that lets you upload multiple files and search across them instantly — directly in the browser.

No backend. No database. No dependencies.

---

## 🚀 Live Demo
👉 [Try It Here](https://your-live-demo-link.com)

---

## 📌 Features

- 📁 Upload multiple text-based files (`.txt`, `.js`, `.json`, `.html`, `.css`, `.md`)
- 🔎 Global search across all uploaded files
- 📄 Line-by-line match detection with line numbers
- ✨ Highlight matching text in results
- 👀 File preview modal with full content view
- 📥 Download original file from preview
- 🧠 Case-sensitive / insensitive search toggle
- 🗑️ Remove individual or all uploaded files
- ⚡ Fully client-side (runs without server)
- 💾 In-memory processing (fast & lightweight)
- 📱 Responsive UI (mobile + desktop)

---

## 🧠 How It Works

1. Upload text-based files
2. Enter search keyword
3. View matching files and line numbers
4. Click **View** to open file with highlighted matches
5. Download or remove files anytime
